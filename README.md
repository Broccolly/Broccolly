## Academic Work
My academic work focused on developing computational techniques to better understand the transition to turbulence in fusion reactors. I was exploring the role recurrent structures such as oscillations and soliton-like waves play in this transition. I predominantly worked in FORTRAN, running high performance code on supercomputers and large computer clusters.

I took additional modules in High Performance Computing (HPC) and Machine Learning.
